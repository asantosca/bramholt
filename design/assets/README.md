# Assets

Exported images, icons, logos, and source design files.

⚠️ Binary files here are **git-ignored** by default (see root `.gitignore`) to keep the
repo small. To version large assets, either:
- enable **Git LFS** (`git lfs track "design/assets/**"`), or
- keep sources in Figma/Drive and list links in `links.md`.

`.gitkeep` and this README are tracked so the folder exists in git.
